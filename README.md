# Benevolence Home Services

Dynamic Next.js marketing site for Benevolence Home Services, built with the App Router, TypeScript, and Tailwind CSS.

## Pages

- `/`
- `/about-us`
- `/services`
- `/careers`
- `/resources`
- `/payment`
- `/contact-us`

## Scripts

- `npm run dev` - start the local development server
- `npm run lint` - run ESLint
- `npm run build` - produce a production build

## Notes

- The design uses the brand palette from the logo reference: blue, orange, and white.
- The navbar and footer are squared and borderless.
- Contact form submissions are handled by `src/app/api/contact/route.ts`.
